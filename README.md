# LAVALINK-VPS

## STEP 1
```js
git clone https://github.com/vaibhav-chauchan/LAVALINK-VPS
```

## STEP 2
```js
cd LAVALINK-VPS
```

## STEP 3 
```js
pm2 start start.sh
```

## EXAMPLE
```js
host: "YOUR VPS IP HERE",
password: "youshallnotpass",
port: 443
```


## IMPORTANT
```js
[+] IF YOU WANT YOU CAN CHANGE YOUR PASSWORD FROM (application.yml)
[-] LAVALINK PORT WILL ALWAYS 443
```
